# 马斯克人设卡（elon-musk）

来源：[https://github.com/alchaincyf/elon-musk-skill](https://github.com/alchaincyf/elon-musk-skill)（MIT）

桌宠运行时只读 `persona-card.json`：
- `system_prompt`：身份、角色规则、5 心智模型、8 启发式、表达 DNA
- `lore.raw_knowledge`：调研索引 + 精炼 references（完整原文在 `references/`）
- `few_shot`：examples/demo-conversation.md 四轮示例

聊天不会直接读旁边的 md；md 仅作源材料备份。
