文字模型

需先安装 Ollama（Windows / macOS 均可，5080 / M4 Pro 由其自动调用 CUDA / Metal 加速）。 保存后本 App 会自动探测并尝试拉起 Ollama 服务；首次使用可从 hf-mirror / ModelScope 导入 GGUF，或点「下载 / 更新模型」。

元元会按昵称记住偏好、约定与对话概要（仅存本机）。清空后她不再记得过往，当前聊天气泡不会自动消失。

视觉模型

Key 仅保存在本机 settings.json，请求经本地代理直连服务商。

需先安装 Ollama。 推荐 minicpm-v:8b（约 5GB），支持中英文识图；也可用 llama3.2-vision:11b（英文更准）或 moondream:latest（极轻，中文弱）。

看图模型与文字模型独立：可分别选择云端 / 本地。例如文字走 DeepSeek 云端、看图走本地 Ollama，或反过来。

语音服务

当前为 macOS：本地语音请使用 Qwen3-TTS（该后端也支持 Windows / Linux）。

本地 Qwen3-TTS 为零样本克隆，不训练音色：填入录音后保存并重启语音服务（切换后端或重开 App）即按此录音克隆。留空则使用内置元元参考音色。

朗读走火山 TTS，通话走火山端到端实时语音；音色共用上方 voice_id。

macOS(Apple Silicon)：保存后自动配置 Qwen3-TTS（创建 venv、装依赖、下载 MLX 量化模型到 ~/Library/Application Support/…/voice-runtime，首次需网络、可能数分钟）。
Windows / Linux：走官方 PyTorch 包 qwen-tts（默认 1.7B 模型），先运行 scripts/windows/setup-qwen3-tts.cmd 创建 .venv-qwen3 环境并安装 torch / qwen-tts 等依赖 （NVIDIA 显卡自动装 CUDA 版 torch；RTX 50 系需 cu128）。
启动服务（通话 :19876，朗读 :19976），零样本克隆参考音频，不消耗火山 token。

保存后自动启动 server_cosyvoice.py（通话 :19877，朗读 :19977）。本机 Whisper + DeepSeek，TTS 走通义云端（上方通义 Key），不消耗火山 token。
