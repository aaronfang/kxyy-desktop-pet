语音服务

当前为 macOS：GPU 本地模型（IndexTTS-2 / CosyVoice3）不可用，本地语音请用 Qwen3-TTS（该后端也支持 Windows / Linux）。

本地模型为零样本克隆，不训练音色：填入录音后保存并重启语音服务（切换后端或重开 App）即按此录音克隆。留空则使用内置元元参考音色。

朗读走火山 TTS，通话走火山端到端实时语音；音色共用上方 voice_id。

macOS(Apple Silicon)：保存后自动配置 Qwen3-TTS（创建 venv、装依赖、下载 MLX 量化模型到 ~/Library/Application Support/…/voice-runtime，首次需网络、可能数分钟）。
Windows / Linux：走官方 PyTorch 包 qwen-tts（默认 1.7B 模型），先运行 scripts/windows/setup-qwen3-tts.cmd 创建 .venv-qwen3 环境并安装 torch / qwen-tts 等依赖 （NVIDIA 显卡自动装 CUDA 版 torch；RTX 50 系需 cu128）。
启动服务（通话 :9876，朗读 :9976），零样本克隆参考音频，不消耗火山 token。

保存后自动启动 server_cosyvoice.py（通话 :9877，朗读 :9977）。本机 Whisper + DeepSeek，TTS 走通义云端（上方通义 Key），不消耗火山 token。

Windows + NVIDIA。保存后自动启动 server_indextts2.py（通话 :9879，朗读 :9979）。 安装程序可勾选配置；也可运行 scripts/windows/setup-gpu-voice.cmd。

Windows + NVIDIA。保存后自动启动 server_cosyvoice3_local.py（通话 :9878，朗读 :9978）。 安装程序可勾选配置；也可运行 scripts/windows/setup-gpu-voice.cmd。

本地服务：保存设置后自动启动
